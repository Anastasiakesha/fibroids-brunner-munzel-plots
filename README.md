# fibroids-brunner-munzel-plots
Statistical analysis and visualization of reproductive hormone levels across menstrual cycle phases in women with and without uterine fibroids.  The project uses R to perform phase-wise comparisons with the Brunner–Munzel test and produces publication-ready boxplots with annotated p-values

Данная задача представляет собой визуализацию репродуктивных гормонов, фаз менструального цикла у женщин с миомой матки и без миомы матки.
Задача состояла из трех пунктов. Задача первого пункта (папка Task1), представляет собой общую статистику по двум группам - европеоиды, азиаты. Задача второго пункта (папка Task2) предстваляет собой статистику по группе европеодов. Третий пункт (папка Task3) представляет собой статистику по группе азиатов.
Для расчёта используется тест Бруннера–Мунцеля (p-value).~
